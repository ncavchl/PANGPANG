# PANGPANG : 뇌 빼고 하는 투표 🐧🚀

> **누르고, 보고, 끝. 1초 컷 투표 종결자 팡팡.**
> 고민은 배송만 늦출 뿐, 결론은 PANGPANG이 내줍니다.

## ✨ 서비스 소개

아직도 단톡방에서 의미 없는 논쟁 중인가요? 답정너 친구 때문에 고통받고 계신가요?
PANGPANG은 결론이 필요한 모든 순간을 위한 **가장 빠르고 힙한 투표 서비스**입니다.
뇌 빼고 툭, 1초 만에 투표하고 친구들에게 공유하세요!

- **압도적 속도**: 3초 만에 투표 생성, 1초 만에 참여 완료.
- **실시간 결과**: 투표 즉시 반영되는 짜릿한 결과 그래프.
- **간편한 공유**: 링크 하나로 카톡, DM 어디든 전파 가능.

---

## 🛠 기술 스택

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (Database, Auth)
- **Deployment**: Vercel

---

## 🚀 시작하기

### 설치

```bash
# 의존성 설치 (pnpm 권장)
pnpm install
```

### 환경 변수 설정

`.env` 파일을 생성하고 아래 내용을 입력하세요:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 로컬 실행

```bash
pnpm dev
```

### 프로젝트 빌드

```bash
pnpm build
```
